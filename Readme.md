# Apple ThumbSlider

This project is an image slider much like the ones you see on Apple's website and apps. It uses thumbnail navigation as well as autoscroll

* Features/Technologies: 
  * HTML5, CSS3 
  * jQuery, JavaScript
  * Custom animate() method
